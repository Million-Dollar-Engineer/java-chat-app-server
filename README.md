# Java Chat Application (Server Side)

## Tech Task
- Spring Boot
- JDBC
- PostgresSQL
- Socket


## How to run this project?

- ###  Run with Maven
- Install packages with Maven
```
    ./mvnw clean install
```
- Run the application with Maven
```
    ./mvnw spring-boot:run 
```
- ###  Run with Docker
```
    docker-compose up -d
```
